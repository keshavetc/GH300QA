  • GitHub Copilot Enterprise - For enterprises ($39/user/month)
Each tier builds upon the previous with additional features and capabilities.

Can GitHub Copilot Individual users access content exclusion features?
No. Content exclusion features are only available in:
  • GitHub Copilot Business
  • GitHub Copilot Enterprise
Individual plan users cannot configure organization-level content exclusions. This is an organizational feature requiring admin-level management, which Individual plans don't support.
What is the minimum number of seats required for GitHub Copilot Business?
GitHub Copilot Business requires a minimum of 1 seat. There is no enforced minimum purchase quantity, making it flexible for organizations of any size.

Key points:

  • Pro-rated billing for partial months

Can organizations mix GitHub Copilot Business and Enterprise licenses?

What are the key decision factors when choosing between Business and Enterprise?
Choose Business if:
  • Primary need is IDE code completion and chat
  • Budget-conscious (half the cost of Enterprise)
  • Team size is small to medium (<100 developers)
  • Don't require mobile or GitHub.com access
  • 90-day audit logs are sufficient
Choose Enterprise if:
  • Need organization-wide knowledge base integration
  • Require mobile app access for developers
  • Want GitHub.com chat and PR summaries
  • Need model fine-tuning for proprietary frameworks
  • Require 180-day audit retention
  • Have GitHub Enterprise Cloud already
  • Large development team (>100 developers)
Consider conducting trials of both to assess actual value.
